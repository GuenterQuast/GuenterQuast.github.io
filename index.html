List of github repositories

by Günter Quast
  • picoCosmo:  Record and analyze data from muon detectors with a picoScope USB oscilloscope

  • picoDAQ:  data aquisition with PicoScope USB oscilloscope

  • PhyPraKit:  Python-Module für das Physikalische Praktikum am KIT

  • mimoCoRB:  multiple-in multiple-out configurable Ring Buffer

  • redpitaya-daq:  Data acquisition with RedPitaya for gamma spectroscopy

  • ppBilliard:  p(roton) p(roton) Billard: video analysis of colliding balls and interpretation as a result of a proton-proton collision

  • PhyPiDAQ:  Software package for data acquisition with Raspberry Pi

  • kafe2 :  Karlsruhe Fit Environment 2: likelihood-based parameter estimation
